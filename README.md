# Reversi ⚪⚫
Javascript implementation of the [Reversi strategy board game](https://en.wikipedia.org/wiki/Reversi).

## How to play ❓
### LOCAL 🏠
Simply run index.html

### ONLINE 🌐
Click [here](https://thecsmine.github.io/ReversiJS).

## Why ❔
To learn HTML, CSS and JS
